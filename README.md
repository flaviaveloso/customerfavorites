# CUSTOMER FAVORITES

## Description

The purpose of this project is allow customers to set theirs favorites products

## Environment

Please make sure that [Node.js](https://nodejs.org/en/)(>= v14) is installed on your operating system.

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
